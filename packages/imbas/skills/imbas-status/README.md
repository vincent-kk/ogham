# imbas-status

현재 또는 과거 imbas 런의 상태를 조회하고, 중단된 런을 재개할 수 있는 읽기 전용 스킬.

## 개요

imbas 런의 진행 상황(단계별 상태, 매니페스트 요약, 블로킹 이슈)을 확인한다.
상태 파일을 수정하지 않는 순수 읽기 전용 스킬이다.

## 서브커맨드

| 커맨드 | 설명 |
|--------|------|
| (기본) | 가장 최근 런의 상태 표시 |
| `list` | 모든 런의 요약 테이블 (Run ID, Phase, Status, Created) |
| `<run-id>` | 특정 런의 상세 상태 (단계별 상태/결과, Epic, Story 등) |
| `resume <run-id>` | 중단된 런의 재개 안내 (다음 실행할 명령 제시) |

## 상태 표시 기호

| 기호 | 의미 |
|------|------|
| ✓ | 완료 |
| ● | 진행 중 |
| ○ | 대기 중 |
| ✗ | 실패/블로킹 |

## 사용 도구

| 도구 | 출처 | 용도 |
|------|------|------|
| `run_get` | imbas MCP | 특정 또는 최근 런의 state.json 읽기 |
| `run_list` | imbas MCP | 프로젝트의 전체 런 목록 |

Jira 도구나 에이전트 스폰 없음. 로컬 상태만 조회.
