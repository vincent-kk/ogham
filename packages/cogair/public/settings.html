recency factor

auto recommends source-checking for time-sensitive facts. strict requires search-verification with citations or an "unverifiable" fallback. gemini has web-tool access by default, so auto is recommended for recency.

sandbox backend

Sandbox backend is only configurable when Sandbox is enabled.

Sets GEMINI_SANDBOX. auto lets gemini-cli pick; sandbox-exec is macOS only.

At least one provider must be enabled.

Default model

Alias applied when start_conversation omits the model. When the matching env var is unset, codex's high / mid / low behave the same as auto (codex-cli default).

Session TTL

Hours before idle sessions are pruned.

CLI spawn timeout

Hard timeout (ms) for each codex / gemini CLI invocation. Hung processes are killed and surface as error.code: network.

Artifacts

Mirror successful responses to disk as markdown files with YAML front-matter. Useful for PR attachment and session review. artifact_path is added to the envelope on success.
