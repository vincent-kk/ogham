# Phase 0 — 종합 판정 (2026-07-23)

> 규칙별 micro-test 결과의 종합. 표본 원문은 각 `phase0/s*.md`. 이 문서는 **결론과 방법론적 발견**만 담는다.

## 핵심 발견 — micro-test의 유효 영역

무지침 단일샷 대조군은 **모든** 규칙의 가치를 측정하지 못한다. 두 부류로 갈린다.

| 부류                                 | 예                                             | 단일샷 대조군                 | 왜                                                                                                                                                                                                                         |
| ------------------------------------ | ---------------------------------------------- | ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **함정 구조의 신규 구조 규칙**       | S1(배선 표지)·S2(export 큐레이션·wildcard)     | **실패 재현 O** — 깨끗한 반전 | 위반이 산출물의 관측 가능한 흔적이고, 올바른 기본 행동(형제 미러링)이 오히려 실패로 이어지거나(S1) 실제 경향성(과다 export·wildcard)이 있다                                                                                |
| **프로세스 규율·좋은 관행의 이관본** | S3 §1(fail-first)·S4 §1(reuse)·S5 §3(grab-bag) | **실패 미재현** — 0/N         | "이 버그 고치고 테스트 추가" 같은 **집중된 단일샷**에서 유능한 모델(claude·codex)은 규칙 없이도 올바르게 행동한다. 규율이 무너지는 건 **다단계 세션에서 부하가 쌓일 때**이며, 그건 단일샷이 아니라 실하니스 A/B로 측정된다 |

**함의**: 프로세스 규칙의 0/N은 "규칙 무용"이 아니라 **계측기 부적합**이다. S3 §1은 두 설계(명시적 mutation·중첩 mutation)로 확인했고, S4·S5도 같은 양상. 이 부류의 통과 근거는 **filid 운영 실적(이관본) + 실하니스 10이슈 A/B**(TODO "최종 효능" 축)이지 단일샷 flip이 아니다.

**중단 조건 재해석**: "절반 이상이 대조군 실패 미재현 → 규칙 세트 재설계"는 **신규 규칙**에 적용된다. 신규 규칙 S1(seiri의 존재 이유)·S2가 강한 반전을 보였으므로 세트는 유효. 이관본의 0/N은 중단 조건을 촉발하지 않는다.

(2) 상시 예산=배포 규칙 1벌+스킬 description ~1줄(본문은 발화 순간에만 로드) — I10 이중 지불 불성립, 규칙은 저장소별 opt-in이라 비용도 선택한 곳만 부담.
