# Nonlinear Heteroscedastic

## 파일

- `data.csv`: 비선형 관계와 이분산성이 함께 있는 연속형 원자료입니다. 각 행은 독립 관측치 1개입니다.

## 통계적 특성

- `x_value`와 `y_value` 관계는 직선이 아니라 완만한 곡선과 주기 성분이 섞인 형태입니다.
- `x_value`가 커질수록 오차 분산이 증가하도록 설계했습니다.
- 몇 개의 명시적 이상치가 포함되어 회귀 진단과 강건 회귀 테스트에 적합합니다.
- `batch`는 x 범위와 강하게 연동되어 있어 단순 집단 비교에 주의가 필요합니다.

## 권장 테스트

- 선형회귀 잔차 진단, 다항/스플라인/GAM 비교, Breusch-Pagan류 이분산 검정, robust regression.
- 산점도와 잔차 플롯 생성 품질 확인.
