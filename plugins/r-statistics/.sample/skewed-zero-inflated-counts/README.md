# Skewed Zero-Inflated Counts

## 파일

- `data.csv`: 영과잉 카운트와 오른쪽 꼬리가 긴 지출액을 포함한 원자료입니다. 각 행은 사용자 1명입니다.

## 통계적 특성

- `visit_count`는 0이 많이 발생하고, 노출 수준이 높을수록 평균 카운트가 증가합니다.
- 일부 큰 카운트가 섞여 과분산과 이상치 영향을 함께 테스트할 수 있습니다.
- `spend_usd`는 방문 수가 0이면 0이고, 양수일 때는 로그정규형 오른쪽 왜도를 가집니다.
- 노출군 크기는 일부러 불균형하게 두었습니다.

## 권장 테스트

- Poisson/negative binomial 회귀, zero-inflated 모델 적합성 판단, 비모수 검정, 로그 변환 전후 비교.
- 평균만 보고하는 분석이 왜도와 영과잉을 놓치는지 검증.
