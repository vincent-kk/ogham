# infra — 프로젝트 설정

## Purpose

프로젝트 로컬 설정의 읽기·쓰기. 현재 담는 것은 개입 강도 다이얼 하나뿐이다.

## Structure

- `configLoader/` — 다이얼 2계층 (`config.json` 기준선 + `runtime.json` 밸브)
- `index.ts` — barrel

## Conventions

- 설정 키를 늘리기 전에 **"이 값의 진실을 누가 소유하는가"** 를 먼저 답한다.
  저장소가 소유해야 할 값(검증 명령·임계치)은 여기 두지 않는다. 사본은
  드리프트하고, 드리프트한 사본은 없느니만 못하다.
- 파일시스템이 이미 답하는 것(어떤 규칙이 배포됐는지)도 두지 않는다.

## Boundaries

### Always do

- 읽기 실패를 정상 상태로 다룬다 — 설정이 없는 프로젝트가 기본값이다.

### Ask first

- 새 설정 모듈 추가.

### Never do

- 오라클 값 보유.
- 세션 훅에서 쓰기.
