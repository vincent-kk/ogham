# filenameSlug

## Purpose

문서 파일명 슬러그 생성. 자연어 제목/태그를 안전한 단일 세그먼트로 정규화하고 UTF-8 바이트 예산으로 절단한다.

## Boundaries

### Always do

- 경로 구분자(`/`, `\`)는 하이픈으로 치환해 flat 단일 세그먼트 보장
- 함수는 호출당 단일 flat 세그먼트만 보장; subdir 분리는 호출자가 세그먼트별로 호출(예: maencofCreate의 명시적 filename)
- UTF-8 바이트 기준 절단, 토큰(하이픈) 경계 우선
- 절단 발생 시 결정적 해시 접미사로 mid-token 절단·충돌 회피

### Ask first

- 바이트 예산·해시 길이 상수 변경

### Never do

- 슬러그 내부에 경로 구분자 보존 (디렉토리 중첩 유발)
- 비결정적 출력 (같은 입력은 항상 같은 슬러그)
