# @ogham/deilen

Claude 가 생성한 markdown 문서를 가독성 있는 브라우저 페이지로 렌더링하고, 라인 단위 피드백을 Claude 에 되돌려줍니다.

## 무엇을 하나

Claude 가 긴 markdown 문서를 만들면, 터미널의 텍스트 벽 대신 `deilen` 이 깔끔한 로컬 웹 페이지(127.0.0.1)로 보여줍니다:

- **가독성 렌더링** — 표·코드 하이라이트·Mermaid·수식. 무거운 렌더러는 lazy-load 라 미사용 시 메모리를 차지하지 않습니다.
- **라인 단위 피드백** — 라인을 선택해 코멘트와 이미지(클립보드 붙여넣기 포함)를 달아 제출하면 Claude 에 자동으로 전달됩니다.
- **원본 복사** — 원본 markdown 을 전체·섹션·코드블록 단위로 복사.

## 동작 원리

로컬 MCP 서버(`tools`)가 문서를 렌더링해 브라우저로 열고, bounded long-poll 로 피드백을 기다립니다. 코멘트와 이미지는 구조화된 텍스트 + Claude 가 실제로 볼 수 있는 이미지 블록으로 되돌아갑니다.

## 사용법

```
# 현재 문서를 페이지로 띄우고 피드백 수집
/deilen:present

# 로컬 설정 UI (테마·자동 오픈·타임아웃·렌더러)
/deilen:setup
```

## 문서

설계 명세는 [`.metadata/deilen/`](../../.metadata/deilen/) 에 있습니다.
